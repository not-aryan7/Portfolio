
# Portfolio Website

Live Demo: https://aryansportfolio1-tau.vercel.app/

A single-page portfolio built with React and Tailwind CSS. It highlights my background with sections for experience, projects, skills, education, and contact details, all styled with a purple-toned theme.

## Features
Responsive navigation with a mobile hamburger menu and anchored section links.
Hero banner with clear calls to action for contacting and browsing projects.
Structured sections for About, Experience, Projects, Skills, Education, and Contact, all using reusable Tailwind utility classes.
Custom color palette defined in `tailwind.config.js` for the primary, secondary, dark, and light theme colors.

## Tech Stack
React 18 (Create React App)
Tailwind CSS 3 with PostCSS and Autoprefixer

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
   The app will be available at the local URL printed in the terminal (default: http://localhost:3000).
3. Create a production build:
   ```bash
   npm run build
   ```
